# Wrench

The Wrench is the primary tool for interacting with logistics nodes.

## Modes

Cycle wrench modes by sneak-scrolling. The current mode is shown in the tooltip and action bar.

- **Wrench** — Right-click a node to open its configuration. Shift-right-click to remove a node. Nodes render as visible overlays while holding the wrench.
- **Copy/Paste** — Right-click a configured node to copy its channel setup to the wrench clipboard. Right-click another node to paste. Ctrl-right-click to paste to all connected nodes sharing the same block.
- **Mass Placement** — Right-click blocks to select them, then open the placement screen to place nodes in bulk with a pre-configured clipboard template.

## Clipboard

The wrench stores a clipboard configuration that can be edited directly. The clipboard records channel settings, filters, and upgrades. When pasting, the required filter and upgrade items are consumed from your inventory and replaced items are returned.
