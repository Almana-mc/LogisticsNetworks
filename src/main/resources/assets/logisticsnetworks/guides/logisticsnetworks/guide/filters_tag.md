# Tag Filter

> Tag filtering is now also available per-entry in the filter detail page. Ctrl+Left Click a base filter slot to access it.

Tag Filter stores one tag value.

You can set it by:

1. Typing a tag
2. Using the extractor slot and picker list

Target type can be:

1. Items
2. Fluids
3. Chemicals

Chemical tag checks are used only when Mekanism is present.
